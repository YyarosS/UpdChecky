# UpdChecky
Это курсовая работа студента ИКБО-30-24, по тема 
"Разработка системы автоматического отслеживания новых версий программного обеспечения". 
Приложение на данный момент предназначено и работает только на свободных и открытых операционных системах, основанных на ядре Linux.

##  Описание

**UpdChecky** — это графическое приложение (GUI), разработанное на Python с использованием библиотеки ```CustomTkinter```. Оно предназначено для централизованной проверки наличия обновлений для программ, установленных через различные пакетные менеджеры в вашей системе, такие как  ```apt, dnf, pacman и Flatpak```

## Требования
Для корректной работы программы, дистрибутив linux должен уметь работать  с такими  пакетными менджерами, как **apt**, **dnf**, **pacman** и **flatpak**.

## Установка
Сначала клонируйте репозиторий проекта на ваш компьютер с помощью git:
```git clone https://github.com/YyarosS/UpdChecky.git 
cd UpdChecky```

после этого запустите файл  ```main```:
```./"main"```
